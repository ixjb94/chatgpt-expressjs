## Pre Install
You need API_KEY,    
For that you can go to https://beta.openai.com/account/api-keys    
And create an API_KEY.

## Installation
```
1- git clone https://github.com/ixjb94/chatgpt-expressjs

2- cd chatgpt-expressjs

3- npm install

4- npm start

5- go to http://localhost:3000
```
